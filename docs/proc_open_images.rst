Process Open Images Script
===================================================

This `script <https://github.com/OlivierNDO/indoor_object_detection/blob/master/data_processing/proc_open_images.py>`_ is used to read images from URLs and their metadata, writing results to a Google Cloud Storage Bucket.

The data_processing directory with all data processing scripts is located `here <https://github.com/OlivierNDO/indoor_object_detection/tree/master/data_processing>`_.
